# Node.js + Express + MongoDB API

API RESTful para gestión de tours turísticos desarrollada con Node.js, Express y MongoDB.

## 🛠️ Tecnologías

- Node.js
- Express.js
- MongoDB / Mongoose
- Postman

## ✨ Características

- CRUD completo de tours
- Filtrado, ordenación y paginación
- Agregaciones MongoDB para estadísticas
- Validaciones con Mongoose
- Middlewares personalizados
- Variables de entorno
